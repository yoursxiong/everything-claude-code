# 构建与修复

增量修复 TypeScript 和构建错误：

1. 运行构建：`npm run build` 或 `pnpm build`

2. 解析错误输出：
   - 按文件分组
   - 按严重程度排序

3. 针对每个错误：
   - 显示错误上下文 (前后各 5 行)
   - 解释问题
   - 提议修复方案
   - 实施修复
   - 重新运行构建
   - 验证错误已解决

4. 停止条件：
   - 修复引入了新错误
   - 重试 3 次后同一错误依然存在
   - 用户请求暂停

5. 显示摘要：
   - 已修复的错误
   - 剩余的错误
   - 引入的新错误

为了安全起见，请一次只修复一个错误！
