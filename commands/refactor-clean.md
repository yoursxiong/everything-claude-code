# 重构清理 (Refactor Clean)

安全地识别并移除冗余代码，并通过测试验证：

1. 运行代码冗余分析工具：
   - knip: 查找未使用的导出和文件
   - depcheck: 查找未使用的依赖项
   - ts-prune: 查找未使用的 TypeScript 导出

2. 在 `.reports/dead-code-analysis.md` 中生成全面报告

3. 按严重程度对发现结果进行分类：
   - 安全 (SAFE)：测试文件、未使用的工具函数
   - 谨慎 (CAUTION)：API 路由、组件
   - 危险 (DANGER)：配置文件、主入口点

4. 仅提议安全删除项

5. 在每次删除之前：
   - 运行完整的测试套件
   - 验证测试通过
   - 应用变更
   - 重新运行测试
   - 如果测试失败则回滚

6. 显示已清理项目的摘要

在运行测试之前，切勿删除代码！
